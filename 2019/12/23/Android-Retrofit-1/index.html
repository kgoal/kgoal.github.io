API 接口的编写

上述代码的作用是定义了一个含有 login 方法的接口，并且该方法使用了 @GET 进行注释，其中传入的参数是 API 的地址。该方法的返回值是一个 Call 类型的变量。

其他注解

上述使用到 @GET 的注解来表示该请求为 GET 请求，同样的，也提供了以下注解来表示请求方法

  • @POST
  • @PUT
  • @HTTP

除了请求方法外，也提供了关于请求参数的和标记的注解

  • @FormUrlEncoded
  • @Multipart
  • @Streaming
  • @Header
  • @Headers
  • @Body
  • @Field
  • @FieldMap
  • @Part
  • @PartMap
  • @Query
  • @QueryMap
  • @Path
  • @Url

具体每个注解的作用自行查看官方文档
